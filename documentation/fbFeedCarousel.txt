User Requirement:
To display first 5 FB feed on homepage using Carousel. Using FB feed SDK library.

URL For reference:
FB SDK - https://developers.facebook.com/docs/javascript/

Files locations For home page carousel

For modifying html and style of the component
Html & CSS template - ./layouts/partials/widgets/carousel.html

To manipulate the Javascript function
JS Function File -  ./layouts/partials/widgets/carousel.html

To add new newsfeed to the website
JSON Files - ./static/files/fbFeed.json

Go to the above file location. There will be a array of JSON. Add the URL to the top of the array to be displayed on the newsfeed. It also affects the feed in the newsFeed page.
